# Manga Processor

マンガのアーカイブファイルを展開し、巻ごとに整理された ZIP 形式に再圧縮して指定のディレクトリ（Google Drive など）へ転送するスクリプトです。

## 主な機能

- **多形式対応**: `.rar`, `.zip`, `.7z`, `.cbz`, `.cbr` などのアーカイブを展開。
- **階層構造の処理**: アーカイブ内のネストされたアーカイブや画像フォルダを自動認識し、適切に ZIP 化。
- **順次処理**: 実行するたびにソースディレクトリから 1 つのタイトルを選択して処理し、完了後に `Done` フォルダへ移動します。

## 必要条件

- **Python 3**
- **unar**: アーカイブ展開用コマンドラインツール
  - macOS: `brew install unar`

## 設定

`process_one_by_one.py` 内の以下のパスを、自身の環境に合わせて編集してください。

- `SOURCE_ROOT`: 未処理のマンガが置かれているディレクトリ
- `TARGET_ROOT`: 処理後の ZIP ファイルを保存するディレクトリ（Google Drive など）
- `TEMP_ROOT`: 一時的な作業ディレクトリ
- `DONE_DIR`: 処理が完了した元ファイルを移動するディレクトリ

## 使い方

1. 設定を確認し、必要に応じてパスを書き換えます。
2. スクリプトを実行します。

```bash
python3 process_one_by_one.py
```

実行のたびに 1 つのタイトルが処理されます。
